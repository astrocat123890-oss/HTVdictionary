# HTV Dictionary

A simple HTML-based dictionary tool.

## Overview

This project is a self-contained HTML dictionary application. Open `index.html` in any modern web browser to use it — no build step or server required.

## Usage

1. Clone or download this repository.
2. Open `index.html` in your browser, or
3. Deploy it via GitHub Pages or Vercel for a live link.

## Deployment

This repo can be deployed for free using:
- **GitHub Pages** — Settings → Pages → Deploy from `main` branch, root folder.
- **Vercel** — Import this repo at [vercel.com/new](https://vercel.com/new) and click Deploy.
